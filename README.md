# LangChain Agent: Chat with Search

This project is a **Streamlit-based conversational agent** that can interact with the web using **LangChain tools**. It allows users to ask questions and get answers from **Arxiv**, **Wikipedia**, and **DuckDuckGo Search** with reasoning steps displayed in real-time.

---

## Features

- Interactive chat interface using Streamlit.
- Supports web search using **DuckDuckGo**.
- Supports scientific article queries via **Arxiv API**.
- Wikipedia knowledge retrieval.
- Shows agent reasoning, thoughts, and actions interactively.
- Uses **ChatGroq** as the language model backend.
